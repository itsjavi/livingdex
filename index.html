Pokmon Living Dex

0.2.0-beta
Living Dex is a visual guide for organizing Pokmon HOME boxes.
This page contains a view of all storable Pokmon forms, including all gender differences.
The project is still in a very early state, but please feel free to send any feedback directly via Twitter
